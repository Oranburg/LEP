1. **The Babel/Nehemia framing is correct.** All three independent agents reading three different projects (AotV, magnifica-babel, the existing LEP page) converged on the *societas / mutuum* + Babel / Nehemia structural opposition. This is the spine.
2. **The Nehemiah Asymmetry is original scholarship.** The beit-midrash verification stood up: classical Jewish commentary is thin on Nehemiah-as-governance-parable. Levin's "Nehemiah Option" (2024) is the modern bridge voice; Zakheim's *Statesman and Sage* (2016) is the sustained Jewish scholarly treatment. The asymmetry holds.
3. **The portfolio is more coherent than the file layout suggests.** Six projects (LE, AotV, JP, FABG, LEP, SoL) bear directly on the engagement. They are not redundant; each carries a distinct lens. Cross-pollination work was already underway — the LEP magnifica page already cites AotV, and AotV §15 already cites MH §140.
4. **Three projects are submission-ready** (AotV, SC, SoL), each needing only narrow finishing work (cite verifications, affiliation updates).
5. **The infographics carry argument visually.** Five official Vatican infographics are in both LawJ and LEP with JSON metadata. They are a visual asset the essays can reference.
